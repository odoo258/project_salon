======================================
Medicament Sales - Brand/Generic Links
======================================

This module adds links that connect the website product pages of brand-name 
medicaments to equivalent generics and vice versa. Medicaments are connected 
based on GCN, and their website visiblity is respected.

Installation
============

To install this module, simply follow the standard install process.

Configuration
=============

No configuration is needed or possible.

Usage
=====

Install and enjoy.

Known Issues / Roadmap
======================

  products and may cause issues with this module.
